# 개발환경 세팅

## 서버 Setup
사용 IDE: Visual Studio Code

1. Node.JS 설치   
   https://nodejs.org/en/ (2024-08-17 기준 v20.16.0 설치함)
2. Nodemon 설치
   - npm i nodemon -D
3. Babel 설치
   - npm i @babel/core @babel/cli @babel/node @babel/preset-env -D
4. ExpressJS 설치
   - npm i express
5. Pug 설치
   - npm i pug
6. 실행 테스트
   - npm run dev
